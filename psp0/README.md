---------------
- Node
- Karma

Instalación
--------
- Instalar _Node_ y _npm_
- git clone
- cd dir/
- `npm install`

Dependencias Globales
------
 `npm install -g karma-cli phatomjs`

Despliegue
----
`node index.js`

Pruebas
------
`karma init test/tests.conf.js`
